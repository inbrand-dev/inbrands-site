Track what you build, avoid, or monitor.

InTrack is a personal habit and behavior tracking app for Android. It helps you create trackers, log progress, view streaks, review completion rates, and see your history on a calendar.

Use InTrack for habits you want to build, behaviors you want to avoid, or patterns you simply want to monitor over time.

Core features

  • Create positive, negative, or neutral trackers
  • Log progress using duration, rep count, checkmark, or occurrence tracking
  • View streaks, completion rates, and calendar history
  • Review multiple trackers from a dashboard calendar
  • Freeze trackers to preserve stats
  • Export and import JSON backup files
  • Use optional local PIN lock protection
  • Set local reminders with Android notifications

Local-first privacy

InTrack stores data locally on your device. It does not require an account, does not use cloud sync, does not use analytics, and does not transmit your tracking data to a server.
